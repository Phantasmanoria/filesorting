********************************************
filesort.pl                    
********************************************

このプログラムには複数の初期設定が存在します。
ただし、起動時にfilesortと同じ場所にconfig.txtがある場合、
その中に記述されている設定で実行します。
また、config.txtがない場合は新規で作成されます。
設定の変更は<<config.txtについて>>を見てください。



<<使用方法>>
Perlでfilesort.plを実行すると起動します。
以下の中で好きなモードを選んでください。(config.txtで固定ができます)


1:拡張子モード
拡張子別に振り分けをします。大文字小文字は区別します。

2:サイズモード
ファイルの大きさで振り分けをします。

3:日時モード
ファイルの作成日時または最終更新日時で振り分けをします。

4:ファイルリストモード
inputdirectoryに入っているファイルの中を一覧で表示します。
左からファイル名、拡張子、サイズ(大きさ自動)、作成日、最終更新日です。

5:コンフィグリストモード
現在読み取っているconfig.txtの場所と内容を一覧で表示します。
config.txtに該当する設定の記述がない場合は空白になります。
また、config.txtを反映させた最終的な設定の内容も表示します。
簡易診断機能付きであり、問題がある場合は該当箇所に<ERROR>が表示されます。


9:終了モード
何もせず終了します。


1~3のモードはつなげて入力することでより細分化して分類することが可能です。
(12だと拡張子>サイズなので同じ拡張子のフォルダの中でも更にサイズで分類される)

classification completed.と表示されたら分類完了です。中身を確認してください。






<<config.txtについて>>
config.txtの中身はそれぞれ以下のようになっています。
シングルクォート(')で囲んで設定を記述してください。
空文字('')の場合は初期設定を読み込みます。
config.txtに該当する設定の記述がない場合も初期設定を読み込みます。
毎回実行する度に標準入力をしたい場合は『//each time//』と記述してください。
なお、初期のconfig.txtではすべて空文字となっています。


#select mode
最初に選択するモードです。初期設定は//each time//(毎回記述)です。
変更する場合は対応する数字を順に連続で記述してください。
記述エラーがあった時は強制的にコンフィグリストモードを実行します。
また、4,5,9を記述した時は確認画面が表示されます。
確認画面で拒否した場合は初期設定を実行します。

#place of input directory
分別対象のフォルダの場所です。初期設定は$HOME/classification/です。
変更する場合は絶対パスまたは相対パスで記述してください。

#place of output directory
分別したフォルダの保存場所です。初期設定は起動時と同じディレクトリです。
変更する場合は絶対パスまたは相対パスで記述してください。

#size of interval(size mode)
サイズモードで使います。
区切るサイズの大きさです。初期設定は//each time//(毎回記述)です。
変更する場合は数字と単位(B/KB/MB/GB/TB/PB)で記述してください。

#which date mode(date mode)
日時モードで使います。
対象とする日時です。初期設定は//each time//(毎回記述)です。
変更する場合は対応する数字を記述してください。

#date units(date mode)
日時モードで使います。
分別の対象とする時間です。初期設定は123(year-month-date)です。
変更する場合は対応する数字を順に連続で記述してください。
振り分け後のフォルダに入力した順番そのままで対象の時間が記入されます。

#date units(filelist mode)
ファイルリストモードで使います。
表示の対象とする時間です。初期設定は123(year-month-date)です。
変更する場合は対応する数字を順に連続で記述してください。
リストに入力した順番そのままで対象の時間が表示されます。

#file command
ファイルを操作する内容を選択します。初期設定は1(コピー)です。
変更する場合は対応する数字を記述してください。

#make log file
ログファイルを作成するか選択します。初期設定はy(作成する)です。
変更する場合はyかnを記述してください。

#date units(result folder)
出力フォルダの名前に記述する時間です。
初期設定は23456(month-date-hour-minute-second)です。
変更する場合は対応する数字を順に連続で記述してください。
フォルダ名に入力した順番そのままで対象の時間が表示されます。
記述エラーがあった時は強制的にコンフィグリストモードを実行します。





<<ファイル構成>>
filesort.pl:メインプログラム。
filesortmode.pl:各モードのサブルーチンプログラム。
filesortdir.pl:ディレクトリを使う操作のサブルーチンプログラム。
(config.txt):設定ファイル。無い場合は自動生成される。
